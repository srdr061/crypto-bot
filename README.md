# Crypto Trading Bot

Binance API kullanarak kripto para analizi yapan servis.

## Özellikler
- Fiyat takibi
- Teknik analiz
- Telegram entegrasyonu

## Kullanım
Servis endpoints:
- `/` - Ana sayfa
- `/test` - Test endpoint
- `/price` - Coin fiyatları

## Teknolojiler
- Python
- Flask
- Docker
- Google Cloud Run
